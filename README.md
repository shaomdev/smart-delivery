# 📦 Smart Delivery — Минималистичный лендинг сервиса умной доставки

Современный, чистый и полностью адаптивный одностраничный сайт для локального сервиса доставки, спроектированный в строгом светлом стиле **Notion / Airbnb**. Проект создан с фокусом на удобство пользователя (UX) и идеальную архитектуру фронтенда.

🚀 **Живое демо проекта:** [Проверить в действии](https://shaomdev.github.io/smart-delivery/)

---

## ✨ Ключевые особенности и UX/UI решения

- **Bento Grid макет:** Секция преимуществ реализована в виде стильной асимметричной сетки карточек разного размера на чистом Flexbox.
- **Двухсторонняя бизнес-логика:** Сайт разделён на два логических потока — калькулятор расчёта стоимости для клиентов и компактная анкета-заявка для привлечения курьеров.
- **Продвинутая кастомизация форм:**
  - Стандартные стрелочки (`spin-buttons`) инпутов веса полностью скрыты во всех браузерах для сохранения строгого дизайна.
  - Уродливые дефолтные выпадающие списки (`select`) заменены на кастомные, а иконка-стрелочка зашита напрямую в CSS через **Base64** для гарантированного отображения.
  - Реализованы кастомные радиокнопки и чекбокс с микро-анимацией плавного появления галочки строго по центру.
- **Независимая архитектура полей:** Все элементы форм жестко связаны через независимую семантическую связку атрибутов `id` и `for` снаружи тегов `label`, что обеспечивает максимальную гибкость верстки и доступность (Accessibility).
- **Идеальный адаптив:** Сайт полностью оптимизирован под мобильные устройства и планшеты. Элементы интерфейса трансформируются в удобные тач-зоны на смартфонах.
- **Эффект матового стекла:** Шапка сайта имеет липкое позиционирование (`sticky`) с размытием заднего плана (`backdrop-filter: blur`), создавая ощущение глубины при скролле.

## 🛠 Технологический стек

- **HTML5** — семантическая разметка (`<header>`, `<main>`, `<article>`, `<footer>`)
- **CSS3** — кастомные переменные (`:root`), Flexbox, продвинутые селекторы состояний (`:focus`, `:checked`, `:hover`)
- **БЭМ (Блок, Элемент, Модификатор)** — строгая и плоская архитектура классов для чистоты и масштабируемости кода
- **Base64 кодирование** — для интеграции векторной графики напрямую в стили
